Fav Movie Watchlist

A web-page that allows the user to search up any movie they like and save it to a watchlist, in a responsive and fashionable design.

Live Link

Project Overview

This web-page was a projects where I wanted to use my React and front-design skills to create a visually pleasing travel journal of places I had visited. But realizing I hadn't visited many crazy places, I decided to create a fantasy travel journal of iconic places I would like to visit.

I decided to go even further by making it a bit more dynamic by connecting the page to GPT-4 API so that the images of places of these iconic locations would be generated by A.I. and if the viewer isn't statisfied with the image, they can click the button to re-generate a new image.

Come view these fantastic places and see if they look like you imagined them to be!
